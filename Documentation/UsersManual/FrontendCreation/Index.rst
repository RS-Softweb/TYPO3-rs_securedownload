Frontend creation
=================

New "Secure Download" Records can be created in the List Module.

Change to a SysFolder and create a new record of type "Secure Download". Insert a title, description and code. Also select or upload
a file. The field codeprompt is not required, but shows a "default value" or "keyword hint" in the input box.

|img-5|

Go now to the page, where the secure download should be shown and create a new plugin record of type "Secure Download".

|img-6|

Select the "Secure Download" record from the SysFolder and save the entry.

|img-7|

The result can look like this in the frontend

|img-3|

After entering the correct code, the input field will be replaced by the download link.
